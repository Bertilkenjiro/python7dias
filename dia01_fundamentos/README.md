# 📘 Dia 1 – Fundamentos

### 📌 Conteúdos estudados
- Variáveis e tipos básicos (`int`, `float`, `str`, `bool`)
- Entrada e saída de dados (`input`, `print`)
- Estruturas condicionais (`if`, `elif`, `else`)
- Operadores relacionais e lógicos (`==`, `!=`, `>`, `<`, `and`, `or`)
